# Learning-Management-System
This project is a frontend and backend development project and is a Learning Management website web application built using EJS, CSS3, Embedded JavaScript(EJS), Node.js, Express.js, and MongoDB. The application is designed only for development & learning purposes. The aim to build this project was to apply my abilities to design a real-world web application. This web app contains a home page, sign-in & sign-up pages, a task manager, a course page for learning, and a logout button.

Deployed project can be viewed by this link : https://lms-website-final.herokuapp.com
