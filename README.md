# Learning-Management-System
This project is a frontend and backend development project and is a Learning Management website web application built using EJS, CSS3, Embedded JavaScript(EJS), Node.js, Express.js, and MongoDB. The application is designed only for development & learning purposes. The aim to build this project was to apply my abilities to design a real-world web application. This web app contains a home page, sign-in & sign-up pages, a task manager, a course page for learning, and a logout button.

Steps :
1) Install Azure tools extension on VS code.
2) Sign in on azure portal and go to the app services.
3) Click on + icon to create new app. A new page opens up.
4) Fill the details like app name, region, etc.
5) in VS code, click on 'A' icon (Azure) and sign in with the same account.
6) Open the main folder in which project folder is stored.
7) Right click on project folder and select 'Deploy to Web App...'
8) Select the same app name in the displayed pop up.
9) The app will auto open in new tab after deployment.
10) Save the URL of app for further use.
